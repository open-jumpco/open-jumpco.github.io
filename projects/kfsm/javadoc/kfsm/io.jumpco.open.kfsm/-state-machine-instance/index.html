StateMachineInstance

class StateMachineInstance<S, E, C, A, R>

This class represents an instance of a state machine. It will process events, matching transitions, invoking entry, transition and exit actions.

Parameters

context - The events may trigger actions on the context of class C

definition - The defined state machine that provides all the behaviour
