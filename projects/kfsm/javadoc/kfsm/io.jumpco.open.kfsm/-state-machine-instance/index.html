class StateMachineInstance<S, E, C, A, R>(context: C, val definition: StateMachineDefinition<S, E, C, A, R>, initialState: S? = null, initialExternalState: ExternalState<S>? = null)

This class represents an instance of a state machine. It will process events, matching transitions, invoking entry, transition and exit actions.
